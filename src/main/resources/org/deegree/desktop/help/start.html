iGeoDesktop: Erste Schritte

Projekt laden

Projekte können entweder durch Mausklick auf den Button in der Toolbar oder über Projekt > Projekt öffnen geladen werden.
Innerhalb des sich öffnenden Dialogfensters, bitte eine Projektdatei *.prj auswählen und auf [Öffnen] klicken.


Neues Projekt anlegen

Neue Projekte können über Projekt > Neues Projekt angelegt werden.
Innerhalb des sich öffnenden Dialogfensters bitte zunächst eine Projekt-Vorlage oder ein existierendes Projekt öffnen, danach das entsprechende Koordinatensystem (EPSG) auswählen. Zusätzlich kann das entsprechende Gebiet entweder via DropDown-Liste (Länderauswahl) oder nach dem Klicken auf [Region auswählen] durch das Aufziehen eines Auswahlrechtecks definiert werden.
Wird hier nichts ausgewählt, wird die BBox automatisch auf den gezeigten Ausschnitt gesetzt.
Abschließend mit [OK] bestätigen.

Bitte Beachten: wird ein bereits existierendes Projekt als Vorlage ausgewählt, werden die darin enthaltenen Ebenen in das neue Projekt übernommen. Dies gilt nicht für das Koordinatensystem - hier wird - on the fly - in das jeweils ausgewählte Koordinatensystem transformiert.
Wird eine Projekt-Vorlage gewählt, enthält das neue Projekt keine Ebenen.
