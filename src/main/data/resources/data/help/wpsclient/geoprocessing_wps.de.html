Web Processing Buffer

Diese Funktionalität beinhaltet das Aufrufen eines vordefinierten Geoprozesses, in diesem Fall von räumlichen Puffern. Mit dieser Funktion können sowohl einzelne/mehrere vorher markierte Objekte einer Ebene, als auch komplette Ebenen bearbeitet werden. Vor dem Aufrufen der Funktion, muß zuerst die betreffende Ebene in der Ebenenansicht markiert und ggf. die betreffenden Objekte selektiert werden, anschließend wird die Funktion   WPS-Buffer (innerhalb der Menüleiste unter > Werkzeuge) aktiviert.
