Fang-Verhalten definieren

Innerhalb diese Reiters kann festgelegt werden, wie sich die einzelnen Ebenen während des Bearbeitungsvorgangs verhalten. Wird die entsprechende Ebene innerhalb des Reiters durch klick mit der LM-Taste selektiert, können auf der rechten Seite die gesetzten Voreinstellungen bearbeitet werden. Das Aktivieren der Fang-Funktion stellt sicher, das benachbarte Geometrien bündig aneinander angrenzen und somit fehlerhafte Topographien vermieden werden.

  • Stützpunkt = Koordinatenpunkt im Verlauf einer Linie / eines Polygon-Teilstücks (Kante)
  • Start-Knoten = Anfangspunkt einer Linie / Kante
  • End-Knoten = Endpunkt einer Linie / Kante
  • Kante = Polygon-Teilstück
  • Fang-Radius = legt die Entfernung fest, ab welcher die neu gesetzten Punkte eingefangen werden sollen
  • Einheit = Pixel oder Karteneinheiten
