Digitalisierung

Unter der Digitalisierung wird allgemeinen der Prozess der Konvertierung analoger Karten und anderer Quellen in eine vom Computer lesbare Form verstanden. Diese Daten werden im Anschluss in Form von Vektordaten gespeichert und/oder zur weiteren Datenverarbeitung verwendet.
Bevor Geometriedaten bearbeitet werden können, muss die entsprechende Ebene in der Ebenenansicht selektiert und editierbar geschaltet sein (es kann immer nur eine Ebene editierbar geschaltet werden).
Das Modul selbst kann über den   Button in der Symbol-Leiste oder innerhalb der Menüleiste über   Werkzeuge > Objektgeometrie Bearbeiten  geöffnet werden.
