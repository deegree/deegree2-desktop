GeoObjekte kopieren

Nachdem einzelne oder mehrere Objekte entweder in der Kartenansicht oder in der Attribut-Tabelle selektiert wurden, können sie kopiert werden:

• als WKT Notierung: innerhalb der Menü-Leiste über Bearbeiten > Objekt(e) als WKT kopieren. Die so kopierten Daten können in andere Bearbeitungs-Programme eingefügt werden.
• als FeatureCollection (*.gml): innerhalb der Menü-Leiste über Bearbeiten > Objekt(e) kopieren oder über den Button . Die so kopierten Objekte können entweder in eine andere Ebene über Bearbeiten > Objekt(e) einfügen oder den Button eingefügt werden oder in ein anderes Bearbeitungs-Programm.
