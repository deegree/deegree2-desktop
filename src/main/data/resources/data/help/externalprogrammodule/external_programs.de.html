Externe Programme starten

Mit Hilfe des Buttons können externe Programme aufgerufen werden, zum Beispiel ein externer Bildbetrachter oder ein Office Programm. Diese Konfigurationen müssen in der *.jnlp Datei vorgenommen werden (siehe hierzu das Dokumentation.pdf). Darüber hinaus müssen die Objekte innerhalb einer Ebene mit den entsprechenden Informationen verknüpft sein.

So können z.B georeferenzierte Bilder als Punktobjekte geladen werden, welche nach dem Selektieren innerhalb der Kartenansicht und dem anschließenden Aktivieren der Funktion in einem entsprechenden Bildbetrachtungsprogramm geöffnet werden. Auf dieselbe Art können Objekte auch mit Office-Programmen verknüpft werden.
