GPS-Daten importieren

GPS-Daten können, sofern sie im *.gpx Dateiformat vorliegen, über Projekt > Neue Ebene hinzufügen > Vektordaten > Datei, über den entsprechenden Button in der Werkzeug-Leiste oder über Projekt > Ebene aus Datei hinzufügen als neue Ebene geladen werden. Die Anwendung erzeugt eine neue Ebene mit den entsprechenden Koordinaten (Punktobjekte).
