Kartenebene neu anlegen

Innerhalb der Ebenenansicht kann über das Kontextmenü eine neue leere Ebenengruppe eingefügt werden. In diese Gruppe können anschließend neue Ebenen geladen oder bereits existierende Ebenen verschoben werden.

Das Anlegen einer einzelnen leeren Ebene ist derzeit nicht möglich. Alternativ können ein bis mehrere zuvor selektierte Objekte aus einer bereits vorhandenen Ebene in eine neue Kartenebene kopiert werden. Diese wird dann der Ebenen- und Kartenansicht hinzugefügt.
