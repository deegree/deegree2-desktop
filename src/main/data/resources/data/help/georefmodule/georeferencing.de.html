Georeferenzierung von Rasterbildern

Vorbereitung

Um das Georefernzierungsmodul nutzen zu können, müssen sie vorher GDAL auf Ihrem Rechner installieren. Wir empfehlen die jeweils aktuelle Version von OSGeo4W unter [http://trac.osgeo.org/osgeo4w].
ACHTUNG! Sie müssen die Systemvariable für das Tool setzten, damit das Modul korrekt funktioniert!

Rasterbilder laden

Die Funktion befindet sich innerhalb der Werkzeugleiste unter Georeferenzierungs Modul öffnen. Nach dem Aktivieren der Funktion öffnet sich ein neues Karten Fenster, sowie das Georeferenzierungs Modul. Zuerst muß das zu georefernzierende Rasterbild geladen werden. Klicken sie auf [Rasterbild laden] und navigieren Sie zum Speicherort des Bildes. Je nach Bildgröße kann der Ladevorgang einige Zeit in Anspruch nehmen.
Wählen Sie nun die Transformationsmethode aus.

Passpunkte bestimmen

Sie können bereits existierende Passpunkttabellen im *.txt Format importieren und erstellte/bearbeitete Passpunkttabellen für eine spätere Verwendung speichern.
Passpunkte neu erstellen = Aktivieren Sie die Schaltfläche [Passpunkte setzten]. Setzten Sie nun den ersten Punkt durch Klicken mit der linken Maustaste in die Referenzkarte - den zweiten korrespondierenden Punkt setzten Sie im Rasterbild. Sobald sie genügend Passpunktpaare (abhängig von der gewählten Transformation) gesetzt haben, können Sie die Transformation durchführen. Sie die Koordinaten auch per Hand in die entsprechenden Felder eintragen, bzw. die Koordinaten dort händisch korrigieren.
